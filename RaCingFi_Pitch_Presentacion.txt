## 2. **Tecnologías y Arquitectura**
**Título:**
🛠️ ¿Cómo está construido RaCingFi?

**Frontend:**
- **React** (Vite + TypeScript + TailwindCSS)
- UI moderna, responsiva y animada
- Integración con TanStack Query para manejo eficiente de datos
- Modelos 3D de autos con **react-three-fiber**
- Desplegado en **GitHub Pages**

**Backend:**
- **Node.js** + **Express**
- **Sequelize** para ORM y gestión de base de datos
- Autenticación segura con **JWT**
- Endpoints RESTful para matchmaking, apuestas, exchange, usuarios, etc.
- Desplegado en **Fly.io**

**Blockchain y Exchange:**
- Simulación de exchange y orderbook en el backend
- Integración con **Solana testnet** para tokens y NFTs
- Marketplace de NFTs y transferencias de tokens

**Herramientas y extras:**
- **Swagger** para documentación de API
- **React Hot Toast** para notificaciones
- **React Select** para inputs avanzados
- **Material Tailwind** para componentes UI

**Visuales sugeridas:**
- Diagrama de arquitectura (Frontend ↔️ Backend ↔️ Blockchain)
- Logos de React, Node.js, Solana, Tailwind, etc.

**Notas para el presentador:**
- Explica cómo cada tecnología aporta a la experiencia y seguridad del usuario.
- Menciona la escalabilidad y facilidad de mantenimiento.
